              OpenAS2 Server
              Version 1.3.3
              RELEASE NOTES

The OpenAS2 project is pleased to announce the release of OpenAS2 1.3.3

The release download file is: OpenAS2Server-1.3.3.zip
The zip file contains a PDF document providing information on installing and using the application.

This release is a bugfix release:
 - Fix comparing sent MIC with returned value since the partner could return the algorithm identifier in different case to what was sent

Java 1.5 or later is required.


Historical list of changes: see the changes.txt file in the release package
