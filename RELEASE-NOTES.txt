              OpenAS2 Server
              Version 1.3.2
              RELEASE NOTES

The OpenAS2 project is pleased to announce the release of OpenAS2 1.3.2

The release download file is: OpenAS2Server-1.3.2.zip
The zip file contains a PDF document providing information on installing and using the application.

This release is a feature enhancement release:
 - Add support for file name filtering by file extension in directory polling module
 - Change the file writing mechanism in the file module to support NFS shares

Java 1.5 or later is required.


Historical list of changes: see the changes.txt file in the release package
